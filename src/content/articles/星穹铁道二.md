# 《崩坏：星穹铁道》系统策划向完整拆解报告

> 报告定位：
> - 用于系统策划岗位面试作品
> - 重点分析核心循环、养成结构、资源经济、日常与版本驱动、商业化联动
> - 分析目标不是“评价游戏好不好玩”，而是拆解其系统如何驱动玩家行为，并识别其优点、问题与可优化空间

# 一、项目基础信息

## 1.1 基础信息

| 项目 | 内容 |
|---|---|
| 游戏名称 | 崩坏：星穹铁道 |
| 类型 | 回合制角色扮演 / 二次元长线运营产品 |
| 平台 | 移动端 / PC / 主机（多端） |
| 商业模式 | 免费下载 + 内购抽卡 + 月卡 + 战令 + 礼包 |
| 核心玩法 | 回合制战斗 + 剧情推进 + 角色养成 + 版本活动 |
| 核心用户 | 喜欢角色收集、剧情体验、低操作门槛战斗与中长线养成的二游用户 |
| 核心竞品 | 原神、FGO、明日方舟、崩坏3、其他二次元长线养成产品 |

## 1.2 一句话总结

《崩坏：星穹铁道》是一款以“低操作门槛的回合制战斗”为核心承载，以“角色收集与多线养成”为长期驱动，并通过“体力资源控制 + 版本角色投放 + 活动更新”维持长线留存的服务型二游产品。

## 1.3 选择该游戏作为拆解对象的原因

- 系统结构完整，适合系统策划岗位分析
- 同时具备核心玩法、养成、资源、任务、活动、商业化等典型模块
- 回合制战斗降低了操作门槛，使系统层分析更清晰
- 其养成系统复杂度适中，适合分析“多线养成如何被玩家接受”
- 版本节奏明确，适合分析“角色投放—资源消耗—版本留存”的联动逻辑
- 面试官通常对该产品较熟悉，便于沟通和追问

# 二、分析目标与拆解方法

## 2.1 分析目标

本报告重点回答以下问题：

1. 《崩坏：星穹铁道》的核心体验由哪些系统共同构成？
2. 它如何通过战斗、资源、养成和版本更新形成长期循环？
3. 多线养成系统为什么没有立刻变成不可接受的复杂负担？
4. 它的日常系统与体力系统如何控制玩家节奏？
5. 当前系统设计的主要优点与主要问题分别是什么？
6. 如果从系统策划角度优化，应优先改哪些模块？

## 2.2 分析方法

- 从玩家路径还原系统目标
- 从资源产出与消耗判断成长节奏
- 从日常、周常、版本内容判断留存抓手
- 从角色投放与养成压力判断商业化联动
- 从新手、中期、后期三个阶段识别主要问题

# 三、目标用户与产品定位

## 3.1 目标用户画像

### 3.1.1 核心人群特征

- 偏好二次元美术、角色塑造、剧情表达
- 愿意接受中长线养成
- 对高操作门槛要求不高，更重视角色搭配与收集
- 愿意参与版本更新，跟随新角色、新剧情持续回流
- 付费意愿主要来自角色获取、角色喜爱、养成效率与长期陪伴价值

### 3.1.2 核心需求

- 角色收集与角色情感连接
- 清晰且有反馈的成长路径
- 低门槛但有一定策略性的战斗体验
- 稳定的版本更新与长期目标
- 日常负担不宜过重，但需要维持“上线有事做”

### 3.1.3 用户分层

| 用户类型 | 主要特征 | 系统诉求 |
|---|---|---|
| 新手用户 | 以剧情和角色吸引入坑 | 上手简单、养成清晰、快速建立目标 |
| 轻度用户 | 每日上线时间有限 | 日常轻量、资源获取稳定、养成反馈明确 |
| 核心用户 | 关注深渊/忘却之庭/混沌回忆等高难内容 | 阵容构筑深度、角色强度、养成效率 |
| 收集型用户 | 对角色与光锥兴趣高 | 抽卡预期明确、角色塑造足够强 |
| 付费用户 | 愿意为角色、效率、收藏付费 | 新角色吸引力、养成补足、长期价值感 |

## 3.2 产品定位

### 3.2.1 产品本质

这款游戏并不依靠高强度操作形成核心壁垒，而是通过以下三层体验结构成立：

- 第一层：剧情演出与角色包装，负责拉新与情绪价值
- 第二层：回合制战斗与弱点击破机制，负责提供可理解的策略体验
- 第三层：角色、光锥、遗器、行迹等养成系统，负责提供长期目标与付费空间

### 3.2.2 差异化优势

与同类二游相比，《崩坏：星穹铁道》的差异化主要在于：

- 回合制降低了操作门槛，提高了更广泛人群的接受度
- 战斗结构相对简洁，但保留了属性、弱点、速度轴、能量、秘技等中层策略
- 日常时长控制相对克制，减轻长期运营产品常见的疲劳问题
- 角色塑造与版本叙事结合紧密，增强角色抽取动机
- 多端覆盖提升了高品质内容消费的便利性

# 四、整体系统结构总览

## 4.1 系统组成

从系统策划视角看，《崩坏：星穹铁道》由以下核心系统构成：

- 核心战斗系统
- 角色获取系统
- 角色养成系统
- 光锥系统
- 遗器系统
- 资源副本与体力系统
- 主线与任务系统
- 日常、周常系统
- 高难挑战系统
- 版本活动系统
- 商业化系统
- 社交弱连接系统

## 4.2 玩家主路径总览

玩家进入游戏  
→ 推进主线、体验剧情、熟悉基础战斗  
→ 获得角色、光锥、养成材料  
→ 提升角色等级、行迹、遗器、队伍搭配  
→ 通过体力系统反复刷取所需资源  
→ 挑战更高难度关卡和周期性高难内容  
→ 围绕新版本、新角色、新活动重新设定目标  
→ 形成长期回流

## 4.3 系统定位判断

| 系统 | 主要作用 | 对玩家意义 | 对产品意义 |
|---|---|---|---|
| 战斗系统 | 提供基础游玩体验 | 理解角色差异与策略搭配 | 承接剧情与养成验证 |
| 角色系统 | 提供收集与构筑空间 | 获得角色满足感与阵容变化 | 形成核心内容更新与商业化抓手 |
| 养成系统 | 提供长期成长目标 | 明确变强路径 | 形成长线留存与资源消耗 |
| 资源副本系统 | 控制成长节奏 | 提供稳定目标 | 维持日常活跃与体力消耗 |
| 高难系统 | 检验养成与阵容 | 提供挑战目标 | 延长中后期生命周期 |
| 活动系统 | 提供新鲜感与限时目标 | 获得奖励与变化体验 | 弥补版本间空窗并促回流 |
| 商业化系统 | 提供付费入口 | 满足收集、效率与身份表达 | 实现长期营收 |

# 五、核心玩法循环拆解

## 5.1 分钟级循环

单次战斗或单次副本的基础循环如下：

进入关卡  
→ 观察敌方弱点与机制  
→ 选择技能、终结技、战技点分配  
→ 触发击破、增益、减益、追击等效果  
→ 完成战斗  
→ 获得结算奖励  
→ 回到养成或下一场战斗

### 分析

该循环的关键特点不是操作强度，而是“低输入频率 + 清晰反馈 + 中度策略”。  
其优点在于：

- 认知成本低，容易理解
- 适合移动端碎片时间
- 适合角色差异化展示
- 易于承接自动战斗

其限制在于：

- 短循环的爽感主要依赖角色特效、数值反馈和阵容配合，而不是玩家手法
- 一旦最优解固化或自动战斗过强，主动操作价值会下降

## 5.2 日循环

典型日循环如下：

上线  
→ 领取登录/邮件/日常奖励  
→ 清理体力  
→ 刷取当前最缺资源  
→ 完成每日任务  
→ 视情况推进主线、活动或高难内容  
→ 投入角色养成  
→ 为下一次上线保留目标

### 分析

《崩坏：星穹铁道》的日循环设计核心，是“轻量但不断线”。

优势：
- 日常耗时相对可控
- 体力消耗目标明确
- 自动战斗降低重复操作负担
- 日常奖励与活跃目标清晰

风险：
- 当主线和活动内容消耗完毕后，日循环容易退化成“上线清体力”
- 如果当前没有强烈的养成目标，日常驱动力会下降
- 轻量是一种优点，但也可能削弱用户深度沉浸时长

## 5.3 周循环

典型周循环包括：

- 每周模拟宇宙相关内容
- 周常活跃或任务目标
- 周期性高难玩法结算准备
- 新角色养成计划的阶段推进

### 分析

周循环承担的不是高频活跃，而是“中期目标整理”。  
它让玩家从“今天刷什么”转向“这周我要把谁练起来、要不要冲更高层”。

## 5.4 版本循环

典型版本循环如下：

版本前瞻或预热  
→ 新角色曝光  
→ 玩家规划星琼与资源  
→ 版本上线  
→ 参与新主线、新活动、新地图探索  
→ 决定是否抽取新角色  
→ 投入新一轮养成  
→ 消耗版本内容后进入相对平稳期  
→ 等待下一版本刺激

### 分析

版本循环是本作真正的长期引擎。  
如果说日常系统负责维持习惯，那么版本循环负责提供情绪峰值与付费转化。

其核心逻辑是：

- 角色投放创造目标
- 活动内容提供短期参与理由
- 新剧情强化角色情感连接
- 养成压力形成资源缺口
- 资源缺口与抽卡决策共同促成回流和付费

## 5.5 核心循环总结

《崩坏：星穹铁道》的循环结构可以归纳为：

- 短期：战斗反馈与资源获取
- 中期：角色养成与阵容成型
- 长期：版本角色与活动更新

这套设计的优点是目标清晰、门槛较低、节奏稳定；  
问题在于中后期过度依赖版本刺激，一旦版本中段内容消耗完毕，玩家容易进入“清体力维持在线”的低波动阶段。

# 六、核心战斗系统拆解

## 6.1 战斗系统构成

核心元素主要包括：

- 回合制行动顺序
- 属性弱点与弱点击破
- 战技点管理
- 终结技能量管理
- 角色定位分工
- Buff / Debuff / 追击 / 持续伤害等机制
- 速度轴、行动提前、控制等衍生策略

## 6.2 系统目标

从系统策划视角，这套战斗系统的目标是：

- 降低操作门槛，扩大用户覆盖
- 保留一定策略空间，避免纯数值碾压显得单调
- 让角色差异化更容易被感知
- 让角色投放可以围绕机制创新而非纯数值膨胀
- 为自动战斗提供天然兼容性

## 6.3 为什么这套战斗系统成立

### 6.3.1 门槛低

- 基础输入少，技能数量不复杂
- 敌人弱点展示清晰
- 行动顺序直观
- 主流阵容搭配可以被普通玩家快速理解

### 6.3.2 深度够用

虽然其战斗深度不属于极高层级，但有足够多的中层变量：

- 弱点击破带来的差异收益
- 战技点的资源分配
- 终结技释放时机
- 速度调轴与角色出手顺序
- 角色间的机制联动，如追击、追加攻击、持续伤害、召唤等

### 6.3.3 与商业化兼容

回合制的另一个优势是：  
角色性能差异更容易通过技能文本与实战结果被玩家理解，因此角色投放的价值更容易被感知和转化。

## 6.4 战斗系统问题

### 问题一：主动操作价值在中低压场景下容易被自动战斗削弱

- 大量资源本、低压战斗中，自动战斗可以代替主动决策
- 玩家会更关注结果效率，而不是过程乐趣
- 久而久之，核心战斗可能更多承担“结算工具”而不是“持续乐趣来源”

### 问题二：部分策略深度对普通玩家不够显性

- 速度、行动提前、机制覆盖率等深层策略较难在前中期被理解
- 玩家容易转向“抄作业”，而不是建立自己的系统认知

### 问题三：部分高难内容对角色覆盖要求较高

- 当高难环境明显偏向新机制角色时，玩家的策略空间会被角色池限制
- 这会让“阵容理解问题”与“角色持有问题”混在一起

## 6.5 战斗系统优化建议

- 在中期教程或活动中强化“机制演示”，让玩家更容易理解速度、击破、增益覆盖等关键概念
- 提供更明确的战后分析信息，如伤害占比、击破贡献、战技点使用效率，强化学习反馈
- 在非高压场景保留自动战斗便利，但增加少量主动决策奖励，避免全过程过度托管
- 控制高难环境的“新角色定向适配感”，避免玩家强烈感知为环境逼抽

# 七、养成系统拆解

## 7.1 养成模块构成

《崩坏：星穹铁道》的角色养成不是单线，而是“多线并行”：

- 角色等级
- 角色突破
- 光锥等级
- 光锥突破
- 行迹升级
- 遗器获取
- 遗器强化
- 阵容搭配与配速调整

## 7.2 养成系统目标

- 构建长期成长路径
- 提供资源消耗池
- 拉开角色从“抽到”到“成型”的时间距离
- 增加不同玩家的效率差异
- 为商业化提供时间与资源价值锚点
- 创造阵容构筑和中后期优化空间

## 7.3 成长路径分析

典型成长路径如下：

抽到角色  
→ 升角色等级与突破  
→ 配套提升光锥  
→ 开始点行迹  
→ 配置可用遗器  
→ 提升遗器等级  
→ 针对高难内容进一步优化词条、速度与队伍搭配

该路径有两个明显阶段：

### 第一阶段：可用化
目标是“能上场”。

特点：
- 角色等级、基础行迹、基础光锥更重要
- 遗器只要主词条和套装勉强可用即可
- 玩家获得的是显著的战力提升感

### 第二阶段：精细化
目标是“打得更稳、更快、更高分”。

特点：
- 遗器副词条、速度轴、队伍联动开始占比上升
- 成长边际递减明显
- 投入增加，但单次提升感受变弱

## 7.4 为什么多线养成没有立即失控

从系统设计角度看，本作的多线养成能被接受，原因在于：

### 7.4.1 阶段性目标清晰

玩家通常知道自己下一步要做什么：

- 缺等级就刷经验
- 缺突破就刷突破材料
- 缺行迹就刷行迹材料
- 缺毕业程度再去刷遗器

虽然线多，但优先级顺序相对明确。

### 7.4.2 早中期爽感足够高

前期角色等级、行迹、基础装备带来的提升很明显，玩家不会一开始就直面遗器随机系统的挫败感。

### 7.4.3 复杂度被版本节奏稀释

由于玩家不可能同时把所有角色全部毕业，系统实际上默认玩家做取舍。  
这种取舍反而构成了长期运营产品中的合理目标延展。

## 7.5 养成系统主要问题

### 问题一：新角色成型成本高

玩家抽到新角色后，往往面临：

- 角色等级材料不足
- 行迹材料不足
- 光锥可能不匹配
- 遗器需要重新刷或重新分配

这会导致“抽到很开心，但短期不能用”的断层。

### 问题二：遗器系统随机性强，后期挫败感显著

遗器系统的问题不在于“需要刷”，而在于：

- 获取随机
- 主词条随机
- 副词条随机
- 强化成长随机

这种多重随机容易造成高投入低反馈。

### 问题三：多角色养成压力与版本角色投放形成叠加

新角色频繁投放对产品是利好，但对玩家而言会造成：

- 资源长期紧缺
- 想练的人多，能练好的少
- 旧角色未完成，新角色又出现
- 中度玩家容易陷入“永远差一点”的状态

## 7.6 养成系统优化建议

| 问题 | 影响 | 优化建议 | 预期结果 |
|---|---|---|---|
| 新角色成型慢 | 抽到即闲置，快乐延迟 | 提供版本角色定向预养成材料包或活动预置资源 | 缩短抽到到可用的距离 |
| 遗器随机过强 | 高投入低反馈，后期疲劳 | 增加更明确的定向养成手段或副词条修正渠道 | 降低毕业挫败感 |
| 多线养成压力大 | 中度玩家容易放弃追求 | 优化材料整合与目标指引，提供更明确的“推荐养成优先级” | 降低理解成本和资源焦虑 |

---

# 八、资源与经济系统拆解

## 8.1 主要资源结构

| 资源类型 | 主要用途 | 获取方式 | 系统意义 |
|---|---|---|---|
| 体力相关资源 | 刷取养成材料 | 自然恢复、道具补充 | 控制成长节奏 |
| 信用点/基础货币 | 多种养成消耗 | 日常、任务、副本 | 通用消耗池 |
| 角色经验材料 | 提升角色等级 | 经验副本、活动 | 基础成长线 |
| 光锥经验材料 | 提升光锥等级 | 副本、活动 | 装备成长线 |
| 行迹材料 | 提升技能与天赋 | 指定副本、活动 | 角色深度成长线 |
| 遗器 | 构筑与后期优化 | 遗器本、活动等 | 后期长期投入线 |
| 星琼/抽卡资源 | 角色与光锥抽取 | 日常、活动、充值 | 商业化核心资源 |

## 8.2 资源闭环

日常与体力消耗  
→ 获得养成资源  
→ 投入角色与队伍培养  
→ 通关更高内容  
→ 获取更高价值奖励或更多资源  
→ 继续投入养成  
→ 为下一个角色或版本做资源储备

## 8.3 体力系统的设计作用

体力系统是本作中非常关键的节奏控制器，主要承担：

- 限制单日成长速度
- 稳定玩家日活
- 引导玩家每日上线
- 放大资源选择的取舍感
- 与版本角色养成需求形成自然绑定

换句话说，体力系统并不只是限制玩家，它实际上在维持整个产品的长期消费节奏。

## 8.4 资源卡点设计分析

### 合理之处

- 不同养成线有不同材料，形成阶段目标
- 日常刷取目标清晰，玩家容易规划
- 通过周本、体力和高难奖励差异，让玩家形成短中长期计划

### 风险之处

- 对新角色、多个新角色或新体系成型不够友好
- 中度玩家长期处于资源不足状态，容易产生无力感
- 高随机系统与体力门槛结合时，会放大刷取挫败

## 8.5 经济系统健康度判断

总体上，本作经济系统是健康的，原因是：

- 资源投放与消耗关系明确
- 成长速度可控
- 养成深度足够支撑长线
- 体力限制避免了过快消耗内容

但它也存在一个典型风险：

> 当高价值追求过度集中到遗器这种高随机系统时，后期“花时间不一定有结果”的感受会逐渐侵蚀成长满足感。

## 8.6 优化建议

- 在后期阶段增加更可控的定向遗器优化手段
- 对版本新角色提供更友好的“成型加速包”式系统支持，但要控制破坏长期节奏的风险
- 增加养成规划辅助，如一键查看某角色成型所需资源总量与缺口来源

---

# 九、任务系统、日常系统与活动系统拆解

## 9.1 任务系统作用

任务系统主要承担：

- 主线推进引导
- 新系统解锁
- 日常上线目标提供
- 奖励发放
- 维持用户轻度活跃习惯

## 9.2 日常系统分析

本作的日常系统相对克制，不属于极重负担型。  
典型特征：

- 每日目标简单明确
- 可与体力消耗路径重叠完成
- 自动战斗降低重复操作感
- 完成门槛不高

### 优点

- 降低流失风险
- 适合学生、上班族等轻中度用户
- 有利于长期维持“不断线”习惯

### 问题

- 日常变化少
- 在版本中后期容易显得机械
- 对高投入玩家来说，日常本身缺乏新鲜度

## 9.3 活动系统分析

活动系统在《崩坏：星穹铁道》中承担的不是纯奖励发放，而是版本期中最重要的新鲜感补充器。

主要作用包括：

- 提供版本中期内容填充
- 用限时奖励促进参与
- 尝试轻度变体玩法
- 强化新角色或新机制的展示
- 维持社区讨论度

### 活动设计优点

- 通常有较强包装感与剧情表达
- 奖励对资源焦虑用户具有吸引力
- 能有效缓解纯日常期的枯燥

### 活动设计问题

- 部分活动玩法深度有限，偏一次性体验
- 活动结束后沉淀价值不高
- 活动与主成长系统的关系有时偏弱，像“临时娱乐内容”

## 9.4 系统策划视角总结

任务系统负责建立习惯，活动系统负责注入波动。  
这两者共同弥补了回合制长线养成产品中，非版本大更新期间体验容易趋平的问题。

---

# 十、高难玩法与后期目标设计

## 10.1 高难玩法的系统作用

高难玩法如忘却之庭、混沌回忆、虚构叙事等，主要承担：

- 检验玩家养成结果
- 放大阵容差异和角色价值
- 形成中后期目标
- 提供周期性挑战与回流动机
- 让“练角色”这件事有明确出口

## 10.2 高难玩法的价值

如果没有高难内容，那么角色养成容易失去验证场景。  
高难玩法的意义不只是发奖励，而是告诉玩家：

- 为什么要练角色
- 为什么要优化遗器
- 为什么要调整阵容
- 为什么要考虑新角色的适配性

## 10.3 高难玩法存在的问题

### 问题一：环境偏向性过强时，容易强化角色焦虑

当高难内容明显有利于某类新角色或新机制时，玩家可能会产生：

- “不是我不会搭配，是我没有角色”
- “不是我没练，是环境不让我用旧角色”

这会削弱挑战的公平感。

### 问题二：高难内容对轻度玩家吸引力有限

轻度用户常常将其视为“额外内容”，而非必要目标。  
这意味着后期目标主要服务核心用户，对广泛用户的留存拉力有限。

## 10.4 优化建议

- 强化高难玩法中的多路径解法，让更多已有角色也能合理通关
- 在挑战失败时提供更明确的问题反馈，如伤害不足、存活不足、节奏问题等
- 提供更多“中难度、但值得参与”的常驻挑战层，缩短轻度用户与核心内容之间的心理距离

# 十一、商业化系统拆解

## 11.1 商业化模块构成

| 模块 | 核心内容 | 面向人群 | 设计作用 |
|---|---|---|---|
| 抽卡 | 角色与光锥获取 | 全体用户 | 核心营收来源 |
| 月卡 | 稳定资源增量 | 中轻付费用户 | 提升长期付费转化 |
| 战令 | 周期性奖励与身份感 | 中度活跃用户 | 强化版本留存与付费 |
| 礼包 | 限时资源或成长补充 | 中高付费用户 | 强化效率购买 |
| 外观/演出类价值 | 相对较弱但存在角色包装价值 | 收集用户 | 补充情绪价值 |

## 11.2 商业化与系统联动逻辑

本作商业化不是单点完成的，而是依赖以下链条：

角色塑造成功  
→ 玩家产生抽取意愿  
→ 抽到角色后进入养成消耗  
→ 养成消耗带来时间与资源压力  
→ 月卡、战令、礼包提升资源效率  
→ 高难环境和版本活动进一步强化角色价值感

也就是说，真正的付费驱动力不是“你必须花钱”，而是：

- 你想要角色
- 你想让角色更快可用
- 你想赶上版本节奏
- 你想在有限时间里获得更好体验

## 11.3 商业化优点

- 核心付费驱动力清晰，主要围绕角色喜爱和阵容价值
- 月卡和战令的梯度较自然
- 商业化与养成系统结合紧密，但通常不直接破坏基本可玩性
- 角色包装和版本叙事强化了抽卡的情绪合理性

## 11.4 商业化风险

- 新角色频繁投放可能让资源不足感长期存在
- 高难环境与角色投放耦合过强时，会放大逼抽感
- 光锥与角色双池结构会提高完成型付费门槛
- 若玩家感觉“抽到也练不起”，会削弱抽卡的即时满足感

## 11.5 优化建议

- 提升“抽到角色后短期可玩”的体验，避免消费快乐被养成门槛稀释
- 降低部分中层付费用户在“角色抽到但养不动”时的挫败感
- 在高难内容中强化旧角色可用性，避免“环境绑定新池”感受过强

---

# 十二、新手引导与前期体验分析

## 12.1 新手阶段的系统目标

新手期主要不是追求深度，而是完成以下任务：

- 让玩家认可世界观和角色
- 让玩家理解基础战斗规则
- 让玩家快速获得前几次成长快感
- 让玩家感受到“抽角色是有价值的”
- 让玩家建立长期目标预期

## 12.2 前期体验优点

- 角色演出和剧情包装容易建立第一印象
- 回合制操作简单，教学成本低
- 早期成长反馈强，战力提升感明显
- 主线推进与系统解锁节奏相对顺滑
- 多端体验增强了内容消费的完整度

## 12.3 前期体验问题

- 当系统逐步展开后，养成线数量开始增加，理解成本抬升
- 前期“因为剧情和角色好看而玩”与中期“开始刷资源”的体验落差较大
- 新手未必能尽早理解遗器、配速、机制协同等中层策略

## 12.4 优化建议

- 在中期阶段补足系统性教学，而不只是前期基础教学
- 用更强的“推荐养成路径”降低新手进入多线养成时的迷茫
- 对遗器、速度、击破等内容设计更直观的教学案例

# 十三、数值与成长节奏判断

## 13.1 成长节奏分层

| 阶段 | 玩家感受 | 系统目标 |
|---|---|---|
| 前期 | 升得快、抽得开心、剧情新鲜 | 建立兴趣与留存 |
| 中期 | 开始刷材料、阵容成型、理解系统 | 建立习惯与资源规划 |
| 后期 | 追求毕业、优化队伍、等待版本刺激 | 拉长生命周期与提升付费转化 |

## 13.2 成长曲线特点

### 前期
- 高频奖励
- 高成长反馈
- 目标明确
- 抽到角色容易有正反馈

### 中期
- 开始出现体力与材料限制
- 多线养成并行
- 玩家开始感受取舍与卡点

### 后期
- 边际收益下降
- 遗器与高难优化成为核心追求
- 对版本和新角色的依赖明显上升

## 13.3 数值风险

- 角色体系越来越多后，旧角色价值容易相对边缘化
- 高难环境若过度迎合新机制，会造成“数值环境驱动焦虑”
- 后期成长过于依赖随机系统，会降低玩家对稳定努力的信任感

# 十四、系统优点总结

## 14.1 优点一：核心循环清晰

本作最成功的地方之一，是把“抽角色—练角色—验证角色—等新版本”这条长线逻辑做得非常清楚。  
玩家很少会不知道自己下一步该做什么。

## 14.2 优点二：门槛控制得当

回合制降低了上手门槛，自动战斗降低了重复劳动成本。  
这使它比许多高操作产品更适合做长期陪伴型产品。

## 14.3 优点三：多线养成复杂但不失控

虽然系统不少，但优先级通常可以被理解。  
前中期目标清晰，玩家不会一开始就被后期随机优化压力压垮。

## 14.4 优点四：版本驱动能力强

新角色、新剧情、新活动之间配合较好，角色包装与版本节奏形成了较强的长期驱动。

## 14.5 优点五：商业化与系统联动自然

其商业化主要依赖角色价值、养成效率和长期计划，不完全是粗暴数值售卖，因此接受度相对更高。

# 十五、主要问题诊断

## 15.1 问题一：后期日常目标波动不足

### 现象
主线与活动消耗完后，玩家日常行为高度固定化，容易退化为清体力。

### 原因
- 日常系统变化少
- 高难玩法更新频率有限
- 中后期可持续目标更多依赖遗器优化，而遗器反馈不稳定

### 影响
- 中后期活跃体验趋平
- 玩家容易进入“上线但不投入”的维持状态
- 对版本中段留存不利

## 15.2 问题二：新角色成型门槛偏高

### 现象
抽到角色后需要较长准备期才能真正进入常用阵容。

### 原因
- 多条养成线同时需要补齐
- 材料副本占用体力
- 遗器适配成本高

### 影响
- 抽卡快乐与实战快乐脱节
- 新角色对玩家的即时吸引力被延迟
- 中度用户对角色扩容意愿可能下降

## 15.3 问题三：遗器系统后期挫败感偏强

### 现象
玩家投入大量体力却长期得不到理想收益。

### 原因
- 多重随机叠加
- 后期成长过度集中于此
- 缺乏足够稳定的纠偏机制

### 影响
- 长期成长满足感下降
- 容易形成“投入与回报不成正比”的感受
- 影响版本中后期的持续游玩动力

## 15.4 问题四：高难环境可能放大角色焦虑

### 现象
部分玩家将高难受阻直接归因为角色池不足。

### 原因
- 环境加成与新机制适配明显
- 角色体系更新快
- 高难内容是角色价值验证出口

### 影响
- 新角色池的逼抽感上升
- 老玩家和回流玩家更容易感受到追赶压力

---

# 十六、优化方案设计

## 16.1 优化原则

- 不破坏原有轻量日常优势
- 不破坏版本角色驱动的商业逻辑
- 优先改善“努力无反馈”和“抽到不能玩”这两类问题
- 尽量通过系统层的“导向”和“纠偏”优化体验，而不是简单暴力送资源

## 16.2 方案一：建立“新角色快速可用化”机制

### 方案描述
当玩家在当期版本获取限定角色后，系统提供一个有限度的“可用化支持”：

- 给出定向基础材料支持，足够升到可上场水平
- 提供阶段性试用遗器或临时推荐套装
- 提供版本角色推荐养成清单和资源缺口导航

### 设计目标
- 缩短“抽到”到“能用”的距离
- 保留深度毕业空间，但先让玩家获得使用快乐

### 预期影响
- 强化抽卡后的即时满足
- 提高新角色实际登场率
- 减轻中度玩家的资源焦虑

### 风险
- 若支持过强，可能压缩长期养成价值
- 需严格控制只解决“可用化”，不直接给“毕业化”

## 16.3 方案二：优化遗器后期纠偏机制

### 方案描述
在不破坏长期刷取结构前提下，增强定向优化能力，例如：

- 增加更稳定的主词条定向手段
- 允许有限次数副词条修正或重铸
- 在长期活跃中提供积累型保底资源

### 设计目标
- 降低高随机系统造成的纯挫败
- 提升“持续投入可预期改善”的感受

### 预期影响
- 提高后期成长的稳定性
- 缓解版本中段玩家缺乏明确目标的问题
- 增强努力与回报的一致性

### 风险
- 若定向程度过高，可能过快缩短毕业周期
- 需保留毕业的稀缺性与长线目标价值

## 16.4 方案三：增加中后期常驻中强度目标层

### 方案描述
在高难和纯日常之间增加“中强度常驻挑战层”，特点包括：

- 目标比日常更有策略要求，但比顶级高难压力低
- 提供周期性变化词缀或任务目标
- 奖励不需要过高，但要足够形成参与理由

### 设计目标
- 弥补版本中段体验平缓的问题
- 给轻中度玩家一个更可触达的中期验证出口

### 预期影响
- 扩展中后期玩家的可玩层次
- 提升非核心高难玩家的参与度
- 减少“除了清体力没别的事”的感受

### 风险
- 若奖励过高，会变成新的负担
- 若玩法变化不足，可能仍然会快速被消耗

---

# 十七、面试总结表达

## 17.1 我对该游戏系统设计的核心判断

《崩坏：星穹铁道》的成功，不只是因为 IP、美术和角色包装，而是因为它把一套非常典型的长线服务型产品结构做得足够清楚：

- 用低门槛回合制承接广泛用户
- 用角色收集和多线养成构建长期成长
- 用体力系统维持日常节奏
- 用高难玩法验证养成价值
- 用版本角色和活动更新提供长期刺激

它的核心优势在于“目标清晰、门槛适中、长线稳定”，而主要问题则集中在“后期日常趋平、新角色成型慢、遗器挫败感高”。

- 能识别中后期留存与资源挫败问题

# 十八、最终结论

《崩坏：星穹铁道》是一款结构非常典型、也非常适合系统策划学习的长线产品。  
它的核心不是某一个单点系统特别夸张，而是多套系统之间的配合关系做得较为成熟：

- 战斗系统足够轻，承担入口与验证功能
- 养成系统足够深，承担长期追求与资源消耗
- 体力与日常系统足够稳，承担活跃维持
- 版本角色与活动系统足够强，承担情绪峰值和商业化转化

从系统策划视角看，这款产品最值得借鉴的不是某个单独数值，而是它对“轻门槛、高包装、长线成长、版本驱动”这组矛盾关系的平衡。

它的主要优化方向，则集中在三个问题：

1. 如何缩短新角色抽取后的快乐兑现时间
2. 如何降低后期遗器随机带来的纯挫败感
3. 如何让版本中段的日常体验不只剩机械清体力

如果能在不破坏原有节奏和商业化结构的前提下改善这三点，这款产品的中后期体验会更加完整。
